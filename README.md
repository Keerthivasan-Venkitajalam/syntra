# Syntra

Autonomous M&A due diligence powered by Linkup + GPT-4o.

## Setup
1. Copy .env.example to .env.local and add keys.
2. Install dependencies.
3. Run the dev server.

## Scripts
- npm run dev
- npm run build
- npm run start

## Notes
- Reports are stored in-memory for now.
- Linkup /search and Vercel AI Gateway calls run server-side.
